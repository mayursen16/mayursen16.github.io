## 🚀 Featured Products

### 1. Astatine Nexus 4
A 4-channel smart relay hub designed for high-reliability room automation and remote power control. 
* **Custom Hardware:** Mass-production-ready PCB with integrated SMPS and strict high-voltage isolation slots.
* **Offline Solar Automation:** Calculates precise sunrise/sunset times locally using an onboard DS1307 RTC and complex solar math—no cloud needed.
* **Non-Blocking Architecture:** Custom `millis()` tracking for fluid countdown timers, sleep functions, and asynchronous UI updates.
* **Deep Diagnostics:** Live tracking of Free Heap, CPU frequency, and I2C bus health via a built-in terminal UI.

### 2. MayurIoT Smart Motor Starter
An industrial-grade IoT gateway designed to protect and automate heavy-duty water pumps (deployed in government infrastructure initiatives like *Har Ghar Nal Se Jal*).
* **Live Telemetry & Protection:** Active load monitoring with a programmable "Auto-Cut on Overload" threshold to prevent motor burnout.
* **Dual Network Protocols:** Connects to local routers for NTP cloud-sync (STA mode), with an automatic air-gapped hotspot fallback (AP mode) for remote field operations.
* **System Event Auditing:** Maintains a continuous, time-stamped log of hardware and HTTP triggers for rapid fault isolation.
