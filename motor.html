MAYURIOT

Smart Motor Starter & Industrial UI

Live Telemetry Dashboard

The central operating interface designed for instantaneous motor control and real-time electrical load monitoring.

  • ACTIVE LOAD MONITORING
    Continuously tracks live amperage draw (e.g., 1.6A) directly on the edge, providing instant feedback on pump health.
  • QUICK START TIMERS
    One-touch presets (3m, 5m, 10m) engineered for field operators to quickly initiate safe, time-restricted irrigation cycles.

Smart Overload Protection

Industrial-grade safety algorithms protecting the water pump and electrical lines from high-current draw and dry-run damage.

  • DYNAMIC TRIP CALIBRATION
    Allows administrators to manually set absolute max-current thresholds (e.g., 20 Amps) based on the specific motor rating.
  • HARDWARE HALT
    If the ADC detects an overcurrent spike, the microcontroller instantly severs relay power to prevent coil burnout.

Autonomous Daily Scheduling

Set-and-forget daily routines that run entirely on the hardware edge to automate complex watering schedules.

  • NTP TIME SYNCHRONIZATION
    Pulls exact Indian Standard Time (IST) from global NTP servers, ensuring schedules execute flawlessly down to the minute.
  • MULTI-SLOT ROUTING
    Configure independent Start and Stop times across different periods of the day directly through the mobile interface.

System Event Auditing

A built-in diagnostic terminal that tracks every single state change and network event in real-time.

  • TIME-STAMPED AUDIT TRAIL
    Records exactly when the motor was started, when timers expired, and if the trigger came from the App (HTTP) or a Physical Hardware Button.
  • RAPID FAULT ISOLATION
    Allows field technicians to read the internal state of the microcontroller to quickly diagnose power issues or missed schedules.

Centralized Configuration

A structured, mobile-first administrative hub giving complete control over device parameters and network routing.

  • COLLAPSIBLE ACCORDION UI
    Keeps complex industrial settings neatly organized into distinct categories for easy navigation on small smartphone screens.
  • SECURE ACCESS
    Centralizes critical features like WiFi setup and Hardware Resets into one location, preventing accidental triggers on the home page.

Cloud & NTP Connectivity (STA Mode)

Securely connects the edge device to existing local infrastructure for global access and time synchronization.

  • INTERNET TIME SCHEDULER
    By connecting to the local router, the device achieves precise time-sync required for the autonomous daily schedules.
  • SECURE CREDENTIAL STORAGE
    Safely writes the user's WiFi credentials (SSID and Password) directly to the ESP's non-volatile memory (EEPROM).

Air-Gapped Hotspot (AP Mode)

Ensures the device remains completely operational and accessible even if the internet goes down or no router is available.

  • CUSTOM BROADCAST
    Allows the user to rename the device's internal WiFi network (e.g., 'Mayur's-Pump') and set a custom security password.
  • DIRECT PHONE-TO-DEVICE CONTROL
    Operate the motor starter out in a remote agricultural field with zero cellular signal or internet connection.

Deep Hardware Analytics

Total transparency into the microcontroller's health, memory, and firmware version.

  • MEMORY & UPTIME TRACKING
    Live tracking of Free RAM (e.g., 31 KB) and total system uptime (e.g., 2h 42m) to monitor stability and prevent buffer overflows.
  • DEVICE IDENTIFICATION
    Displays the unique MAC Address and Chip ID, critical for registering the device on commercial MQTT brokers or cloud databases.
